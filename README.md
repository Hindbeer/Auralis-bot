# About 

**Auralis** ([link](https://t.me/auralis_kuve_bot)) - It will help to put your tracks in order via telegram. In just two clicks, you can change the design of the track.

### Features: ([link](https://youtu.be/jfkA_2QS7mM))
- ✅ **Lang** — multi-language support
- ✅ **Audio formats** — Support for various formats
- ✅ **Friendliness** — Simple and user-friendly interface

## Launch

The project uses `python 3.14` and `aiogram 3.x`. Shows how to launch using **UV**.

1. Clone the repository
```bash
git clone https://github.com/Hindbeer/Auralis-bot.git
cd Auralis-bot
```
2. Installing [**uv**](https://docs.astral.sh/uv/)
3. Fill in `.env` 
4. Project launch
```bash
uv run main.py
```

## Launch using Docker

1. Clone the repository
```bash
git clone https://github.com/Hindbeer/Auralis-bot.git
cd Auralis-bot
```
2. Fill in `.env` 
3. Project launch
```bash
docker-compose up --build
```
